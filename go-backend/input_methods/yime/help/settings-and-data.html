设置与数据

这页帮助主要解释 Yime 设置项写到哪里、哪些操作只是保存文件、哪些操作会触发重新构建。

主要目录

  • 用户配置与用户词库：%APPDATA%\PIME\Rime
  • 日志目录：%LOCALAPPDATA%\PIME\Logs
  • 共享运行时数据：go-backend\input_methods\yime\data

桌面语言栏与任务栏入口

  • 桌面语言栏：可直接点击“用户词库”“反查编码”和“工具中心”。
  • 任务栏停靠状态：左键单击“中”按钮，在弹出的“设置”菜单中直接选择上述三个命令。
  • 两处入口功能相同；“工具中心”打开的程序窗口名称仍为“Yime 工具箱”。

输入方案与连续输入

变长、等长、省键三种方案都启用连续输入和 Rime 整句组句。变长、省键派生码保留虚首音，作为自动分词边界。虚首音对应零声母，包括拼音书写中的隔音符号 ' 以及 y、w；实首音则对应非零声母。

候选选择键

  • Space、Enter 或 Shift+1 选择第一候选；Shift+2 至 Shift+9 选择其余候选。
  • 候选窗显示 ⇧1…⇧9，箭头表示必须按住 Shift。与流行拼音输入法不同，Yime 有意不采用裸数字键选词；Base 层 0—9 十个数字键始终用于编码，候选出现时也不切换含义，Shift+0 不选词。
  • 物理键面是 ! @ # $ % ^ & * (，布局图会照实标出；候选窗不直接用连续标点作序号，以保持易读。

语言栏里的维护项

  • 数据维护：集中放置“同步数据…”“重新部署…”和“打开目录”；两项执行型维护命令都会再次确认。
  • 重新部署…：仅在手工修改或恢复配置、排查旧缓存时使用。确认后在后台构建并校验当前方案，再于安全请求边界重建会话；不会恢复默认词库、删除用户词库或重复启动任务。
  • 同步数据…：确认后调用 Rime 自带的同步能力，只处理 Rime 管理的学习数据，不包含 YIME 设置、用户词库源文件和屏蔽词表。
  • 打开目录：打开用户目录、共享目录、同步目录和日志目录，方便检查“文件有没有真的写进去”。

注意：yime_settings_state.json 这类 Yime 独立界面状态不属于 Rime 原生同步范围。

设置工具中的备份与恢复

  • 备份：在 Windows“文档\YIME 备份”中创建带时间戳、文件清单和 SHA-256 校验的可移植用户数据快照。
  • 备份范围：包括 YIME 设置、用户词库、屏蔽词表和 Rime sync 同步数据；不包括可重新生成的 build、临时通知文件，以及输入法运行期间被锁定且不能一致复制的 *.userdb LevelDB 目录。
  • 恢复：恢复最近一次手工备份。恢复前自动保存当前数据，恢复后重新构建 Rime，并通知 YIME 重载设置和词库。

“应用”设置时为什么需要重建

像方案切换、候选项数这类设置，不只是改界面显示，还需要让 Rime 运行时重新读取并编译相关配置。所以单纯保存文件可能不够，通常需要再执行一次重建或重新部署。

  • 保存文件：把设置写到用户目录中的配置文件。
  • 应用：保存后再调用 rime_deployer.exe，让运行时真正看到新值。

独立工具完成重建后，会通知当前 YIME 会话重新读取设置。

用户词库

  • 可从桌面语言栏“用户词库”、任务栏“设置 → 用户词库”或工具箱“词库管理”打开。
  • 添加对话框点击“保存”后可继续添加；“退出”、取消或窗口 X 才结束。
  • 系统初始词库中已有的词条会收到警示并被拒绝添加。
  • 添加和编辑对话框内可输入步进值调整权重，每点击一次“加”或“减”调整一次。
  • 添加、编辑、删除、撤销、应用、导入和错误提示统一使用居中的中文操作按钮，不再显示系统 OK 或 Yes/No。
  • 点击“应用”会为变长、等长和省键三种方案生成词库并部署。

用户屏蔽词表

  • 源文件为 %APPDATA%\PIME\Rime\yime_blocklist.txt。
  • 工具支持添加、搜索、多选删除、导入、导出，以及撤销最近一次成功改动。
  • 保存成功后立即影响后续候选过滤，无需重新部署。

高级布局

  • 从“工具中心 → 高级布局”打开，可从当前布局新建副本、保存多个命名方案并即时试算三模式编码。
  • 个人方案保存在 %APPDATA%\PIME\Rime\yime_layouts；官方共享布局保持只读。
  • 确认应用后，等长、变长、省键三种模式同步重建系统词典和用户自建词表、迁移各自的学习库；全部编译成功后才通知输入会话统一刷新。

工具箱中的目录与帮助

桌面语言栏“工具中心”或任务栏“设置 → 工具中心”会打开 tool-hub.exe，窗口名称为“Yime 工具箱”。

  • 用户数据目录：打开当前账户的设置、词库和 Rime 用户数据。
  • 共享数据目录：只用于查看随安装提供的方案、词典和编码数据，重装时会被覆盖。
  • 查看帮助：打开主帮助；反馈说明：说明如何配合诊断报告和截图反馈问题。
