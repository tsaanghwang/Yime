# Trial Feedback

For the shortest useful feedback, choose the closest case:

- Installation fails.
- Installation works, but the input method does not open.
- The input method opens, but the candidate window does not appear.
- The candidate window appears, but text cannot be committed.
- It works the first time, then fails after restart.
- It mostly works, but a key or interaction feels wrong.

Please also include:

1. Which application you tested in.
2. What you did.
3. What you saw.

You can use Help > Copy trial feedback template from the PIME language bar menu.
