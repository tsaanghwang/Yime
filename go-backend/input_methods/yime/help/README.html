Yime 帮助

这份帮助页面向普通用户，集中说明 Yime 在 PIME 中的常用入口、候选操作、反查显示和独立工具方向。帮助、设置、诊断这些较重的界面都尽量放在独立窗口里，语言栏只保留轻量分发。

语言栏轻量化 独立工具优先 避免宿主不稳定

快速开始

  • 在 PIME 语言栏中切换到 Yime 方案。
  • 输入编码后，用空格、回车或候选序号上屏。
  • 方向键、Page Up、Page Down、Home、End 可用于移动候选。
  • 按 Esc 取消当前组字。

菜单说明

  • 工具：打开独立工具箱，集中进入词库管理、反查编码、系统词库审查、用户屏蔽词表、设置和诊断。
  • 设置：可以切换变长、等长、省键方案，以及中英模式、全半角、标点、候选排列和候选项数。
  • 数据维护：集中放置“同步数据…”“重新部署…”和“打开目录”，避免误点维护命令。
  • 重新部署…：仅在手工修改配置、恢复文件或排查旧缓存时使用。它在后台重新编译并校验当前方案，再于安全的请求边界重建会话；不会恢复默认词库，也不会删除用户词库。
  • 同步数据…：调用 Rime 自带的用户数据同步，只处理 Rime 管理的数据，不包含 Yime 独有的设置、用户词库源文件和屏蔽词表。
  • 显示编码：控制候选上的反查注释显示方式。
  • 用户词库：打开独立词库管理器，适合做增删改查、导入导出、冲突预览和应用词库。
  • 设置工具：应用方案和候选显示设置，也可备份或恢复用户数据。
  • 查看帮助 / 反馈说明：分别打开本页和问题反馈指南。

反查显示

  • 隐藏编码：只保留候选本身。
  • 标准拼音：显示带声调的标准拼音。
  • 音元拼音：显示当前 Yime 方案对应的拼音编码。
  • 键位序列：显示实际键位序列。

数字声调拼音主要用于录入用户词条，例如 lv4、ri4 ben3，不作为日常候选显示模式。

工具箱

  • 词库管理：增删改查、导入导出、撤销和应用个人词条。
  • 反查编码：按词条查询拼音及三种音元编码。
  • 系统词库审查：只读扫描疑似问题词条并导出报告。
  • 用户屏蔽词表：添加、搜索、多选删除、导入导出和撤销屏蔽词。
  • 设置工具：应用设置，并创建或恢复带校验的可移植用户数据备份。
  • 诊断工具：检查路径、进程和日志，并导出反馈报告。

常用路径

  • 用户 Rime 数据：%APPDATA%\PIME\Rime
  • PIME 日志：%LOCALAPPDATA%\PIME\Logs
  • Yime 共享运行时数据：go-backend\input_methods\yime\data
  • 可移植用户备份：文档\YIME 备份
