音元输入法项目概述

音元输入法(YIME)是一种创新的汉语音码输入系统，基于独创的音元分析理论构建

核心特性

更短的编码长度

平均码长比全拼输入法缩短大约百分之三十

更高的输入效率

支持简拼、双拼和并击(合击)多种输入模式

更精的音系表达

通过音元系统精确描述汉语语音系统的特征

多平台支持

提供网页原型和桌面端以及手持端实现

技术栈

技术领域 主要技术
核心引擎 Python 3.10+
Web界面 Node.js 16+, React, TypeScript
测试框架 pytest, Jest
构建工具 npm, webpack
