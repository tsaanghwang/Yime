音元输入法开发指南

本指南为开发者提供音元输入法的开发环境搭建、代码规范和贡献流程说明

开发环境搭建

系统要求

  • 操作系统: Linux/macOS/Windows (WSL2推荐)
  • Python: 3.10+
  • Node.js: 18+
  • Git: 2.30+

环境配置步骤

# 克隆仓库
git clone git@gitcode.net:2302_77644309/Yinyuanxitong.git
cd Yinyuanxitong

# 安装Python依赖
pip install -r requirements.txt

# 安装前端依赖
npm install

# 配置开发环境变量
cp .env.example .env.development

代码规范

Python代码规范

  • 遵循PEP 8风格指南
  • 使用类型注解(Type Hints)
  • 模块级docstring必须包含功能说明和示例
  • 函数级docstring必须包含参数、返回值和示例

TypeScript代码规范

  • 使用ESLint和Prettier进行代码格式化
  • 严格类型检查(noImplicitAny)
  • React组件使用函数式组件和Hooks
  • 接口定义使用I前缀(如IProps)

测试指南

运行测试

# 运行Python测试
pytest --cov=pinyin --cov-report=html

# 运行前端测试
npm test

贡献指南

贡献流程

  1. Fork项目仓库
  2. 创建特性分支 (git checkout -b feature/your-feature)
  3. 提交代码变更 (git commit -am 'Add some feature')
  4. 推送到远程分支 (git push origin feature/your-feature)
  5. 创建Pull Request

发布流程

版本发布步骤

# 更新版本号
npm version patch  # 或 minor/major

# 构建发布包
npm run build
python setup.py sdist bdist_wheel

# 发布到PyPI
twine upload dist/*

# 创建Git标签
git push origin --tags
