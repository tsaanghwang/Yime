音元输入法API文档

本API文档适用于音元输入法核心引擎和前端接口

接口分类

核心引擎API 前端接口API 插件系统API

韵母转换API

YunmuConverter 类

class pinyin.yunmu_to_keys.YunmuConverter

核心韵母转换器，负责将标准拼音韵母转换为音元键位

方法 参数 返回值 说明
convert yunmu_dict: Dict[str, str] Dict[str, str] 批量转换韵母字典
convert_single yunmu: str str 转换单个韵母
get_stats None Dict[str, Any] 获取转换统计信息

干音分析API

GanyinAnalyzer 类

class tools.ganyin_analyzer.GanyinAnalyzer

干音分析器，用于分析汉字干音特征

方法 参数 返回值 说明
analyze hanzi: str Dict[str, Any] 分析单个汉字干音特征
batch_analyze hanzi_list: List[str] List[Dict[str, Any]] 批量分析汉字干音特征
get_feature_names None List[str] 获取所有干音特征名称

前端接口API

InputMethod 类

class src/App.tsx.InputMethod

前端输入法核心接口，负责处理用户输入和候选词展示

方法 参数 返回值 说明
handleInput key: string void 处理用户按键输入
getCandidates None string[] 获取当前候选词列表
selectCandidate index: number string 选择候选词

错误码说明

错误码 说明 解决方案
40001 无效的韵母输入 检查输入是否为标准拼音韵母
40002 转换规则冲突 检查规则优先级设置
50001 干音分析失败 检查汉字是否在字库中
60001 前端输入超时 减少输入延迟或优化性能
