项目来源

音元输入法系统架构

音元输入法采用分层架构设计，核心引擎与前端界面分离

目录结构

Yinyuanxitong/
├── pinyin/            # 核心拼音处理模块
│   ├── constants.py   # 音元常量定义
│   ├── yunmu_to_keys.py # 韵母转换核心逻辑
│   └── ...
├── tools/             # 辅助工具集
│   ├── ganyin.py      # 干音分析工具
│   ├── pianyin.py     # 拼音分析工具
│   └── ...
├── src/               # 前端源代码
│   ├── App.tsx        # 主应用组件
│   └── ...
├── tests/             # 测试代码
│   ├── test_yunmu_to_keys.py
│   └── ...
└── docs/              # 项目文档
    ├── THEORY.md      # 理论文档
    └── ...

模块划分

核心引擎

  • 韵母转换模块
  • 干音分析模块
  • 拼音分析模块
  • 音元映射模块

前端界面

  • 输入法主界面
  • 候选词展示
  • 用户配置管理
  • 词库管理

辅助工具

  • 数据验证工具
  • 统计分析工具
  • 批量处理工具
  • 测试框架

数据流图

用户输入 → 前端界面 → 核心引擎 → 候选词生成 → 前端展示
          ↓                     ↑
      用户配置           词库数据

设计模式

设计模式 应用场景 实现模块
策略模式 韵母转换规则 yunmu_to_keys.py
观察者模式 用户配置变更 src/App.tsx
工厂模式 插件系统 pinyin/rule_plugin.py
装饰器模式 性能统计 tools/orchestrator.py
