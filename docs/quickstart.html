音元输入法快速开始指南

只需5分钟即可完成安装并开始使用音元输入法

环境要求

组件 最低版本 推荐版本
Python 3.10 3.12
Node.js 16 20
npm 8 10

安装步骤

1. 克隆仓库

git clone https://gitcode.net/2302_77644309/Yinyuanxitong.git
cd Yinyuanxitong

2. 安装Python依赖

pip install -r requirements.txt

3. 安装前端依赖

npm install

运行指南

开发模式 生产模式

启动开发服务器

npm run dev

访问 http://localhost:3000 使用输入法

构建生产版本

npm run build
npm run start

示例演示

基本输入示例

你好 → nh
世界 → sj
音元输入法 → yy srf

常见问题

安装依赖时出现错误

确保使用正确版本的Python和Node.js。可以尝试:

python --version
node --version
无法启动开发服务器

检查端口3000是否被占用，或尝试:

npm run dev -- --port 3001
输入法响应缓慢

可能是词库加载问题，尝试:

npm run build
npm run start
