候选词排序规则 (sort_rules.json)

{
  "default_weights": {
    "frequency": 0.6,
    "length": 0.2,
    "recent_use": 0.2
  },
  "custom_rules": [
    {
      "pattern": "^[a-z]+$",
      "weights": {
        "frequency": 0.4,
        "length": 0.3,
        "recent_use": 0.3
      }
    }
  ]
}

此配置文件控制候选词的排序算法，可以调整不同因素的权重来优化输入体验。

环境变量

变量名 默认值 说明
YYIM_DEBUG false 启用调试模式，输出更多日志信息
YYIM_CACHE_SIZE 1000 设置候选词缓存大小
YYIM_MAX_SUGGESTIONS 9 设置最大候选词数量
YYIM_THEME light 设置界面主题(light/dark)

部署配置

生产环境推荐配置

# 生产环境推荐配置
YYIM_DEBUG=false
YYIM_CACHE_SIZE=5000
YYIM_MAX_SUGGESTIONS=5
YYIM_THEME=light

在生产环境中，建议关闭调试模式以提高性能，同时根据实际使用情况调整缓存大小。
