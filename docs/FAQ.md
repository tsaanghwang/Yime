# 常见问题解答（FAQ）

---

## 1. 项目定位与基础

**Q: YIME 是什么？适合哪些用户？**
A: YIME（音元输入法编辑器）是一套以音元为码元的汉语音码输入系统，适合对汉语输入效率、音系表达有较高要求的用户、语言学研究者、输入法爱好者及开发者。

**Q: YIME 与传统拼音输入法有何不同？**
A: YIME 采用音元系统，平均码长更短，输入效率更高，音系表达更精细，支持多种输入模式（全拼、简拼、双拼、并击）。

---

## 2. 安装与环境

**Q: 需要什么环境才能运行 YIME？**
A: 需要 Python 3.14+（核心引擎）、Node.js 16+（Web 前端）、现代浏览器（Chrome/Firefox/Edge）。

**Q: 如何安装和启动？**
A:
1. 克隆仓库：`git clone https://github.com/tsaanghwang/YIME.git`
2. 安装依赖：`pip install -r requirements.txt` 和 `npm install`
3. 启动前端：`npm run dev`，访问 `http://localhost:3000`

---

## 3. 功能与使用

**Q: 支持哪些输入模式？**
A: 支持全拼、简拼、双拼、并击四种输入模式，可在界面或配置中切换。

**Q: 如何自定义词库或添加新词？**
A: 可通过前端界面或调用 API（如 `addUserWord`）添加用户词条，具体见 API 文档。

**Q: 如何切换键盘布局？**
A: 支持美式键盘（重置键位）和中式键盘（扩展键位），可在设置中选择。

**Q: 如何查看或修改音元与键位的映射？**
A: 相关映射表见项目内 JSON 文件和文档，支持自定义修改。

## 4. 开发与贡献

**Q: 如何参与开发或贡献代码？**
A: 欢迎 Fork 仓库，按 [CONTRIBUTING.md](../CONTRIBUTING.md) 规范提交 Pull Request，建议先补全单元测试并通过 CI。

**Q: 有哪些推荐的开发工具和规范？**
A: 推荐使用 VS Code、PyCharm、WebStorm 等，Python 遵循 PEP8，前端遵循 ESLint/TypeScript 规范。

**Q: 如何运行和编写单元测试？**
A: Python 测试用例可用 `pytest`，前端用 `npm test`（Jest）。
