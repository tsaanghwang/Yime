音元输入法使用手册

本手册详细介绍音元输入法的各项功能和使用方法

基本输入方法

音元键位图

音元输入法采用独特的键位设计，将声母和韵母映射到键盘的不同位置，实现高效输入。

输入流程

  1. 输入声母对应的键位
  2. 输入韵母对应的键位
  3. 从候选词中选择目标汉字
  4. 按空格键确认输入

功能详解

连续输入

音元输入法支持连续输入多个音节，系统会自动分词并显示候选词：

输入: woshizhongguoren
输出: 我是中国人

简拼输入

对于常用词组，可以只输入每个音节的声母：

输入: zgr
输出: 中国人

模糊音设置

在配置文件中可以设置模糊音，适应不同地区的发音习惯：

{
  "fuzzy_pinyin": {
    "zh": ["z"],
    "ch": ["c"],
    "sh": ["s"]
  }
}

操作指南

快捷键 功能
空格 确认当前候选词
数字1-9 选择对应编号的候选词
PageUp/PageDown 翻页查看更多候选词
Ctrl+Shift+F 切换全角/半角
Ctrl+Shift+P 切换中英文标点
Ctrl+Space 切换中英文输入

常见场景

输入姓名

对于不常见的人名，可以使用完整拼音输入：

输入: zhangxueyou
输出: 张学友

输入专业术语

对于专业术语，可以使用自定义词库功能：

{
  "custom_words": {
    "jisuanji": "计算机",
    "ruanjian": "软件"
  }
}

故障排除

1. 检查输入法是否已激活 (Ctrl+Space)

2. 检查键盘布局是否正确

3. 重启输入法服务

1. 检查是否开启了模糊音设置

2. 尝试输入完整拼音而非简拼

3. 更新用户词库

1. 删除用户配置文件 (~/.config/yinyuan)

2. 重启输入法
