# 普通用户完整帮助

本文面向普通试用者，说明当前外挂输入法原型怎么启动、怎么用、遇到问题先看什么。

这不是开发文档，也不是系统级 Windows 输入法安装说明。当前主线仍是 Windows 桌面输入法原型。

## 它现在是什么

- 它已经可以在 Windows 外部窗口里做组合输入、显示候选、选词和回贴。
- 它还不是系统级 IME，所以不同应用里的手感和稳定性可能不完全一样。
- 推荐先在记事本、VS Code 普通文本区等简单环境试用。

## 启动方式

推荐入口：

python -m yime.input_method.app

等价入口：

python run_input_method.py

常用参数：

- --copy-only：只复制候选，不自动回贴。
- --font-family "Microsoft YaHei"：指定候选框字体。

## 启动后看到什么算正常

正常情况下，启动后可能看到：

- 键盘监听已启动，按 ESC 退出
- 键盘监听未启用，将使用手动输入模式
- 运行时候选回退到 SQLite runtime_candidates

这些都不一定表示出错，要先结合环境和启动日志判断。

## 基本操作

- 直接在外部窗口输入码元，候选框会在光标附近出现。
- 数字键可选当前页候选。
- Space / Enter 可上屏首选候选，或提交待上屏文本。
- Home / PgUp / PgDn / End 可翻页。
- ESC 用于退出当前组合，Ctrl+Q 用于关闭窗口。
- 待命状态下，可点“音”图标或按当前唤起热键恢复。

## 候选框菜单能做什么

- 设置：调整候选列表、交互方式、前景背景颜色、字体大小、透明度、活动窗置顶。
- 工具 -> 用户词库：加入当前词条、删除当前词条、编辑用户词库、应用用户词库、导入用户词库、导出用户词库。
- 帮助 / 关于：查看这份主说明和当前原型定位。

## 用户词库推荐流程

- 想在本机维护词库时，优先用“编辑用户词库”。
- 编辑并保存后，用“应用用户词库”把修改写回当前词库。
- 如果词库来自另一台机器、备份文件或外部整理结果，用“导入用户词库”。
- 如果要备份当前词库内容，用“导出用户词库”。

## 遇到问题先看什么

- 提示“将使用手动输入模式”：先检查 Python 3.12 和 pywin32 是否可用。
- 候选词为空或结果不完整：先看启动日志确认当前候选来源是 JSON、SQLite 视图还是静态候选表。
- 某些应用里行为不稳定：先回到记事本等简单环境复现，再判断是不是目标应用兼容性问题。

## 当前已知边界

- 这还是原型应用，不是系统级 TSF / IMM32 输入法。
- 不保证所有 Windows 应用里行为完全一致。
- 自动回贴、词频排序和极端交互场景仍在继续打磨。
