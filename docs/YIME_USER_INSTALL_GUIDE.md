# Yime 用户安装指南

本文档面向最终用户，说明如何安装和使用 Yime（音元拼音）输入法。

## 系统要求

- Windows 10 1903 或更高版本
- 管理员权限（安装时需要）

## 安装步骤

### 1. 运行安装程序

双击 `YIME-Setup.exe` 启动安装向导，按提示完成安装。

### 2. 切换到 Yime 输入法

安装完成后：

1. 点击任务栏右下角的语言指示器（如"中"或"ENG"）
2. 选择 **Yime 音元拼音**
3. 或按 `Win + 空格` 循环切换输入法

### 3. 首次使用检查

切换到 Yime 后，如果候选窗口没有出现：

1. 在任意文本框中输入字母（如 `f`），应出现候选词
2. 如果无反应，等待 10-30 秒（首次加载需要初始化 Rime 引擎）
3. 仍无反应，参见[故障排除指南](YIME_TROUBLESHOOTING.md)

## 基本使用

### 编码输入

Yime 使用声韵双拼编码，24 个声母（手音）对应键盘字母，33 个韵母（韵音）对应键位组合。输入编码后从候选窗口选择。

### 候选选择

- 默认使用 `空格`、`-`、`=`、`\`、`` ` `` 五个键选择候选
- `空格` 选第一候选，其余键选第 2-5 候选
- `↑` `↓` 或 `PageUp` `PageDown` 翻页

### 三种编码方案

| 方案 | 说明 | 适合人群 |
|------|------|----------|
| 变长（variable） | 默认方案，编码长度随音节变化 | 新用户推荐 |
| 等长（full） | 每个音节固定 4 键编码 | 习惯固定键位者 |
| 省键（shorthand） | 缩写编码，击键更少 | 追求速度者 |

通过语言栏的"方案"菜单切换。

### 语言栏菜单

右键点击语言栏的 Yime 图标，可访问：

- **工具箱**：词库管理、反查编码、设置等
- **方案**：切换编码方案
- **候选排列**：竖排/横排切换
- **候选数**：调整每页显示的候选数量
- **帮助**：查看使用说明

## 常用工具

### 反查编码

输入汉字查询其 Yime 编码。支持精确匹配和包含匹配，显示标准拼音、数字标调和三种方案的编码。

### 词库管理

添加、编辑、删除用户词库中的词条，设置权重调整候选顺序。添加对话框支持连续保存，退出按钮、取消或窗口 X 结束；系统初始词库已有词条会被拒绝。批量权重可输入步进值后逐次加减。修改后点击"应用"生成并部署三种方案的用户词库。

### 设置工具

调整候选排列方向、反查注释显示模式等。

## 数据目录

| 目录 | 路径 | 说明 |
|------|------|------|
| 用户数据 | `%APPDATA%\PIME\Rime\` | 用户词库、设置、日志 |
| 共享数据 | `C:\Program Files (x86)\YIME\go-backend\input_methods\yime\data\` | 系统词库、编码表 |

**注意**：不要手动修改共享数据目录中的文件，这些文件会在重装时被覆盖。用户自定义内容应放在用户数据目录。

## 卸载

通过 Windows 设置 → 应用 → 已安装的应用 → YIME → 卸载。

用户数据（`%APPDATA%\PIME\Rime\`）不会随卸载删除，如需清除请手动删除。
