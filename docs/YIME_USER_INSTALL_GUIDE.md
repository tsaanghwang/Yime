# Yime 用户安装指南

本文档面向最终用户，说明如何安装和使用 Yime（音元拼音）输入法。

## 系统要求

- Windows 10 1903 或更高版本
- 管理员权限（安装时需要）

## 安装步骤

### 1. 运行安装程序

双击 `YIME-<版本>-setup.exe` 启动安装向导。安装包为 YIME-only，直接安装 Yime Go 后端、原生工具和所需 TSF 组件，不再提供旧 PIME Python/Node 输入法或组件选择页。

### 2. 切换到 Yime 输入法

安装完成后：

1. 点击任务栏右下角的语言指示器（如"中"或"ENG"）
2. 选择 **Yime 音元拼音**
3. 或按 `Win + 空格` 循环切换输入法

### 3. 首次使用检查

切换到 Yime 后，如果候选窗口没有出现：

1. 在任意文本框中输入字母（如 `f`），应出现候选词
2. 如果无反应，等待 10-30 秒（首次加载需要初始化 Rime 引擎）
3. 仍无反应，参见[故障排除指南](YIME_TROUBLESHOOTING.md)

## 基本使用

### 编码输入

Yime 使用 24 个首音类和 33 个乐音类 Yinyuan ID。当前试验布局把它们投影到
47 个 Base 键和 11 个 Shift 键位；输入编码后从候选窗口选择。

### 候选选择

- `空格`、`Enter` 或 `Shift+1` 选择第一候选
- `Shift+2` 至 `Shift+9` 分别选择第 2 至第 9 候选
- 候选窗用 `⇧1`、`⇧2`…`⇧9` 提示必须同时按 Shift；裸数字 `1`…`9` 是编码键，不会选词
- 与常见拼音输入法不同，Yime 有意不采用裸数字键选词；数字键始终用于音元编码，候选出现时也不会切换含义
- 键盘上这些组合的键面字符是 `! @ # $ % ^ & * (`，帮助和布局图会保留键面标注，但不把标点直接用作候选窗序号，以免影响辨识
- Base 层的 `-`、`=`、`\` 已用于编码，不再触发候选选择
- `↑` `↓` 或 `PageUp` `PageDown` 翻页

### 三种编码方案

| 方案 | 说明 | 适合人群 |
|------|------|----------|
| 变长（variable） | 默认方案，编码长度随音节变化 | 新用户推荐 |
| 等长（full） | 每个音节固定 4 键编码 | 习惯固定键位者 |
| 省键（shorthand） | 缩写编码，击键更少 | 追求速度者 |

三种方案均支持连续输入和 Rime 整句组句。变长、省键方案现在保留零声母音节的虚首音编码，用它标明音节边界；这项内部编码规则使它们与等长方案一样能够稳定自动分词。

通过语言栏的"方案"菜单切换。

### 语言栏菜单

使用桌面语言栏时，可直接点击 **用户词库**、**反查编码** 和 **工具中心**；其中“工具中心”打开的程序窗口名称为“Yime 工具箱”。

语言栏停靠在任务栏时，独立按钮可能不会显示。左键点击任务栏上的“中”按钮，在弹出的 **设置** 菜单中可直接访问：

- **用户词库**：打开独立词库管理器
- **反查编码**：打开独立反查工具
- **工具中心**：打开词库管理、反查编码、系统词库审查、屏蔽词表、设置和诊断等独立工具
- **方案**：切换编码方案
- **候选排列**：竖排/横排切换
- **候选数**：调整每页显示的候选数量
- **数据维护**：集中使用“同步数据…”“重新部署…”和“打开目录”

## 常用工具

### 反查编码

输入汉字查询其 Yime 编码。顶部可在查询框与查询按钮之间直接设置“包含匹配”和编码方案；支持精确匹配和包含匹配，显示标准拼音、数字标调和三种方案的编码。

语言栏“反查显示”中的“音元拼音”会在候选注释中显示 PUA 音元字符；“键位序列”仍显示实际 ASCII 按键编码。安装包会安装所需的 YinYuan 字体，这项显示设置不会改变输入方式或用户词库编码。

### 词库管理

添加、编辑、删除用户词库中的词条，设置权重调整候选顺序。添加对话框支持连续保存，退出按钮、取消或窗口 X 结束；系统初始词库已有词条会被拒绝。批量权重可输入步进值后逐次加减。所有操作提示和确认框使用居中的中文“确认 / 取消”等按钮，不再显示系统 `OK` 或 `Yes/No`。修改后点击"应用"生成并部署三种方案的用户词库。

### 设置工具

调整输入方案、候选项数、候选排列方向和反查注释显示模式。点击“应用”会保存设置、重新构建 Rime，并通知当前输入会话刷新。

“备份”会在 Windows“文档\YIME 备份”中创建带清单和 SHA-256 校验的可移植用户数据快照；“恢复”会先备份当前数据，再恢复最近一次手工备份并重新构建。备份包含设置、用户词库、屏蔽词表和 `sync` 数据，不直接复制运行中被锁定的 `*.userdb` 数据库。若需保存 Rime 自动学习词频，请先使用语言栏“设置 → 数据维护 → 同步数据…”。

### 系统词库审查与用户屏蔽词表

- **系统词库审查**只读扫描系统词库，可按规则筛选并导出 JSON/TSV 报告，不会修改系统词库。
- **用户屏蔽词表**支持添加、搜索、多选删除、导入导出和一步撤销，成功改动立即影响后续候选。

### 诊断与反馈

遇到安装、启动、路径或日志问题时，打开“诊断工具”执行检查并导出报告；再从工具箱打开“反馈说明”，按说明附上测试应用、操作步骤、实际现象和截图。分享报告前请检查其中是否包含不希望公开的路径或日志片段。

## 数据目录

| 目录 | 路径 | 说明 |
|------|------|------|
| 用户数据 | `%APPDATA%\PIME\Rime\` | 用户词库、设置、屏蔽词和 Rime 用户数据 |
| 共享数据 | `C:\Program Files (x86)\YIME\go-backend\input_methods\yime\data\` | 系统词库、编码表 |
| 日志 | `%LOCALAPPDATA%\PIME\Logs\` | 后端运行日志 |
| 备份 | `%USERPROFILE%\Documents\YIME 备份\` | 设置工具创建的可移植用户数据快照 |

**注意**：不要手动修改共享数据目录中的文件，这些文件会在重装时被覆盖。用户自定义内容应放在用户数据目录。

## 卸载

通过 Windows 设置 → 应用 → 已安装的应用 → YIME → 卸载。

用户数据（`%APPDATA%\PIME\Rime\`）不会随卸载删除，如需清除请手动删除。
