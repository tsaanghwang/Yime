大段代码注释语法对比

语言 多行注释语法 快捷键(常见IDE)
HTML/CSS Ctrl+/ (VS Code)
JavaScript/TypeScript /* 注释内容 */ Ctrl+Shift+/ (VS Code)
Python '''注释内容''' 或 """注释内容""" Alt+Shift+A (VS Code)
Java/C/C++/C# /* 注释内容 */ Ctrl+Shift+/ (VS Code)
PHP /* 注释内容 */ 或 # 注释内容 Ctrl+/ (VS Code)
Ruby =begin 注释内容 =end Ctrl+/ (VS Code)
Bash/Shell : <<'COMMENT' 注释内容 COMMENT Ctrl+/ (VS Code)

使用建议

  • 大多数现代IDE支持快捷键批量注释/取消注释
  • Python的多行字符串也可作为注释使用
  • HTML注释不能嵌套使用
  • JavaScript/TypeScript中推荐使用JSDoc风格的注释
