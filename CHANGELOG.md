# 更新日志

本文件只记录当前主线中对试用者、使用方式或发布内容有直接影响的变更。

## 2026-05-10

### 修复

- 修正手动输入路径对小键盘数字和小键盘符号的误拦截，避免它们再被当作候选选择或被投影成错误码元。
- 修正 `Shift+Q~P` 这类字面数字在候选框中的显示投影，避免输入后又被重新映射成错误字符。
- 恢复 `` ` ``、`-`、`=`、`\\` 作为第 2~5 候选快捷选择键的行为，避免被前面的修复连带破坏。

### 文档

- 将帮助文档、FAQ、候选框状态提示和试用者单页中的候选选择说明统一为当前规则：
  首选候选用 `Space` / `Enter` / 鼠标左键；
  第 2~5 候选用 `` ` `` / `-` / `=` / `\\`；
  其余候选用方向键定位后再按 `Space` / `Enter`，或直接鼠标左键。
