# 主仓库占位说明：键盘布局链已外置

## 当前定位

主仓库中的这份文件不再充当键盘布局发布线的正式版本说明。

当前 Windows 键盘布局辅助工程已经独立到：

- `C:/dev/Yime-keyboard-layout`

主仓库这里只保留一个过渡占位，防止旧路径突然失效。

## 现在应看哪里

如果要查看键盘布局辅助工程的正式说明，请看：

- `C:/dev/Yime-keyboard-layout/releases/v1.0/CHANGELOG.md`
- `C:/dev/Yime-keyboard-layout/README.md`
- `C:/dev/Yime-keyboard-layout/docs/SHARED_FILE_POLICY.md`

## 主仓库中的角色变化

在主仓库中，键盘布局相关内容现在只应被理解为：

1. 由 IME 运行时和辅助仓库共同消费的布局真源
2. 少量仍保留在主仓库中的过渡入口或兼容脚本
3. 不属于当前 IME 主线的独立发布工程
