# 编码测试

这份清单用于反复手测当前音元输入法原型的候选翻页、首选上屏、分页键和拣字键。

说明：

- `物理键面` 写的是当前布局下实际要按的键。
- 单字编码是 4 键一组。
- 词语编码按音节分组，组与组之间用 ` / ` 分隔，通常按完前一组再接着按下一组。
- 当前运行时词语库里，同码同音词的真实上限是 `5`，没有超过 `5` 或 `9` 的词语页可测；超过 `9` 的翻页测试请用下面的单字编码。
- `AltGr+u` 和 `Shift+j` 这类写法表示需要对应组合键层。

## 最短手测顺序

1. 先用 `yi4` 或 `li4` 测单字大页候选，看翻页键和 `` ` - = \ `` 是否正常。
2. 再用 `zheng4 shi4` 测同音词 5 选，确认 `Space`、`Enter` 和鼠标点击都能一次上屏。
3. 上屏后让 UI 进入半透明静置，再次点亮，重复输入 `bao4 li4` 或 `da4 zhi4`，确认恢复位置和外部焦点都正常。
